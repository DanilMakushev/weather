# Weather App

Настольное приложение для отслеживания погоды, написанное на **C++** с использованием фреймворка *Qt6*.

---

## Как запустить

1. Перейдите в раздел **[Releases](https://github.com/DanilMakushev/weather/releases)** на странице этого репозитория.
2. Скачайте архив `weather_app.zip` для Windows.
3. Распакуйте архив в любую удобную папку (например, на Рабочий стол).
4. **Важно:** Не перемещайте файл `cities.db` из папки, он должен находиться в одной директории с `weather_app.exe`.
5. Запустите `weather_app.exe`.

> 📌 **Примечание:** При работе приложение автоматически создаст конфигурационные JSON-файлы в своей папке для сохранения ваших настроек и истории.

---

## Как получить и настроить API ключ

Для работы службы погоды приложению требуется действующий API-ключ.

1. Зарегистрируйтесь на сайте погодного сервиса (https://openweathermap.org/).
2. Перейдите в личный кабинет в раздел **API Keys** и создайте (скопируйте) ваш уникальный ключ.
3. Запустите приложение `Weather App`.
4. В появившемся окне ввода ключа вставьте скопированный токен и нажмите **Подтвердить**.
